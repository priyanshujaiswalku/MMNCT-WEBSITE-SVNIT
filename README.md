# 🏏 MMNCT Website - SVNIT

A modern web application built for **MMNCT (Manoj Memorial Night Cricket Tournament)** at SVNIT Surat.
This platform enhances audience engagement by providing live updates, fixtures, points table, and interactive features.

---

## 📌 About MMNCT

MMNCT is a **day-night cricket tournament** organized at SVNIT Surat in memory of a student Manoj Kumar.
It started in 2006 to honor his passion for cricket and continues to inspire students.

## 📂 Project Setup (Localhost)

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/priyanshujaiswalku/MMNCT-WEBSITE-SVNIT.git
cd MMNCT-WEBSITE-SVNIT
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run the Development Server

```bash
npm run dev
```

👉 Open in browser:
http://localhost:3000
